# CardioAI: Heart Disease Prediction System

CardioAI is an educational, production-ready AI-powered clinical decision support system designed to predict whether a patient is at risk of coronary heart disease. It utilizes clinical metrics from the UCI Cleveland Heart Disease dataset.

> [!IMPORTANT]
> **Educational Disclaimer:** Predictions generated by the machine learning model in this system are NOT medical diagnoses and should not be considered formal clinical advice. Always consult a certified medical practitioner.

---

## 🌟 Key Features

- **End-to-End Pipeline:** Automated data cleaning, imputation, outlier handling, feature engineering, and training.
- **Multi-Algorithm Comparison:** Evaluates Logistic Regression, Decision Trees, Random Forests, Support Vector Machines, KNN, Gradient Boosting, and XGBoost.
- **Hyperparameter Tuning:** Automates GridSearchCV optimization for the best-performing model.
- **RESTful API Backend:** FastAPI services for single (`/predict`) and batch (`/batch_predict`) inferences.
- **Streamlit Interactive Dashboard:** Features a clean, medical-themed UI (blue/white) containing:
  - **Dashboard:** System metrics and telemetry.
  - **Predict Disease:** Parameter form inputs, risk percentage display, gauge charts, and lifestyle tips.
  - **Dataset Explorer:** Filter, search, and download dataset subsets.
  - **Model Performance:** Metrics, Confusion Matrix, and ROC curves.
  - **Explain Prediction:** Global SHAP beeswarm plots and Local SHAP Waterfall charts.
- **Export Reports:** PDF generation for diagnostic summaries.
- **Database Logging:** Automatically persists prediction inputs and metrics to an SQLite database.
- **Containerized Deployment:** Ready to run with Docker and Docker Compose.

## 🚀 Getting Started

### Prerequisites
- Python 3.12
- Docker & Docker Compose (Optional)

### Installation
1. Clone the repository and navigate to the directory:
   ```bash
   cd heart-disease-ai
   ```

2. Create and activate a virtual environment:
   ```bash
   python -m venv .venv
   # Windows:
   .venv\Scripts\activate
   # Linux/macOS:
   source .venv/bin/activate
   ```

3. Install required libraries:
   ```bash
   pip install -r requirements.txt
   ```

---

## 🏃 Usage

### 1. Data Ingestion & Preprocessing
To fetch the raw UCI dataset and trigger the preprocessing pipeline:
```bash
python fetch_data.py
python -m src.preprocessing
```

### 2. Model Training & Tuning
To compare all classifiers, tune hyperparameters on the best one, and export reports:
```bash
python -m src.train
```

### 3. Running the REST API (FastAPI)
Launch FastAPI server locally:
```bash
uvicorn api.main:app --reload --port 8000
```
Visit Swagger documentation at [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs).

### 4. Running the Streamlit Dashboard
```bash
streamlit run app/streamlit_app.py
```
Open [http://localhost:8501](http://localhost:8501) in your browser.

---

## 🐳 Docker Deployment

To spin up both the FastAPI backend and Streamlit dashboard instantly:
```bash
docker-compose up --build
```
- Streamlit: [http://localhost:8501](http://localhost:8501)
- FastAPI: [http://localhost:8000/docs](http://localhost:8000/docs)
